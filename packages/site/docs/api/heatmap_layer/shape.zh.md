`shape` 方法用于指定热力图层绘制热力的类型，现在支持经典热力、蜂窝热力、网格热力三种类型。

### shape('heatmap')

`shape` 为 `heatmap` 热力图层用于绘制 2D 经典热力。

### shape('heatmap3D')

`shape` 为 `heatmap3D` 热力图层用于绘制 3D 热力。

### transforms

网格热力和蜂窝热力的实现依赖配置 `transforms` 的参数，[具体说明使用](/zh/docs/api/source/source/#transforms)。
