## 简介

单波段栅格数据渲染、根据栅格数据进行可视化着色

## Option

<embed src="@/docs/common/layer/options.md"></embed>

`RasterLayer` 图层主要实现栅格数据的可视化，栅格数据主要来源是卫星遥感数据，如数字高程图，植被分布图，夜光图。
