## 图层控制方法

### show 显示

图层显示

```javascript
layer.show();
```

### hide 隐藏

图层隐藏

```javascript
layer.hide();
```

### isVisible 是否可见

图层是否可见

return `true | false`

### setIndex 设置图层层级

设置图层绘制顺序

### fitBounds 缩放到图层范围

缩放到图层范围

```javascript
layer.fitBounds();
```

### setMinZoom 设置图层最小可见等级

设置图层最小缩放等级

参数

- zoom {number}

```javascript
layer.setMinZoom(zoom);
```
