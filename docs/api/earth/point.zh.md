用户在地球模式下使用点图层无需做额外的操作，L7 会自动识别地球模式并相关的转化

## 使用

地球点图层通过 `PointLayer` 和 `EarthLayer` 实例化

```javascript
import { PointLayer, EarthLayer } from '@antv/l7';
```
