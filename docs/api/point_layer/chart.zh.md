## 使用

使用二维统计图表代替点状符号的一种特殊复合形式，复合图表地图中常用扩展图形还有柱状图、曲线图、玫瑰图等

通过自定义 Marker 实现自定义图表

```javascript
import { PointLayer } from '@antv/l7';
```
