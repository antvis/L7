### 实现

下面我们来介绍如何绘制简单线。

- 你可以在 `L7` 官网上找到[在线案例](/examples/gallery/animate#grid)

```javascript
import { LineLayer } from '@antv/l7';
const layer = new LineLayer()
  .source(data)
  .shape('simple')
  .size(10) // size 方法不生效 线宽始终为 1px
  .color('#f00')
  .style(...)
```

### shape

为了绘制简单线，我们需要将 `shape` 的参数设置成 `simple`。

🌟 简单线图层不支持动画和纹理
