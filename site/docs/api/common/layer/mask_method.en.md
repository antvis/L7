## layer mask method

### addMask

```
Add Mask layer
```

```ts
const polygonLayer = new Polygon();
layer.addMask(polygonLayer);
scene.render();
```

### removeMask

```ts
const polygonLayer = new Polygon();
layer.removeMask(polygonLayer);
scene.render();
```

### disableMask

```ts
layer.disableMask();
scene.render();
```

### enableMask

```ts
layer.disableMask();
scene.render();
```
