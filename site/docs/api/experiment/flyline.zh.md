用户在地球模式下使用飞线图层无需做额外的操作，L7 会自动识别地球模式并相关的转化

## 使用

地球飞线图通过 `LineLayer` 和 `EarthLayer` 实例化

```javascript
import { EarthLayer, LineLayer } from '@antv/l7';
```
